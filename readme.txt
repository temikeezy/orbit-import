Bulk-import users and assign them to BuddyBoss/BuddyPress Streams (Groups) with dry-run, mapping, batching, resumable jobs, and role assignment.

== Description ==
Admin tool to import users from CSV and enroll them to BuddyBoss/BuddyPress groups.

== Installation ==
Upload the `orbit-bulk-user-importer` folder to `/wp-content/plugins/` and activate.
